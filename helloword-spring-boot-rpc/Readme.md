##  简单介绍


## Sender1  相当于主动方应用，负责发起对 被动方接口的调用。给hello1对列发送消息。
## Receiver1 相当于被动方应用（消费hello1对列消息）。 这里将模拟调用远程方法，通过线程等待模拟长时间调用处理。
## Sender2 相当于被动方应用给主动方应用回复远程调用的应答。给hell2队列发送消息
## Receiver2 相当于hello2队列的消费端，处理接收 被动方调用的结果处理

## 以上过程，类似RPC
